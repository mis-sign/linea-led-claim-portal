WhatsApp Notifications (Maytapi)
Sends WhatsApp updates to the customer's number when a claim is filed or its status changes.
This number gets a WhatsApp alert every time a new claim is filed — separate from the customer's copy.
Email Notifications (Resend)
Sends a branded email to the customer and to your Support Email whenever a claim is filed or updated. Uses Resend (resend.com) — a plain internet API, so it works reliably on Render (Gmail SMTP is blocked on many hosts).
Free — sign up at resend.com → API Keys → Create API Key, then paste it here.
Only fill this in once you've verified your own domain on resend.com. Until then, leave blank — emails still send fine from Resend's shared address.
Photo Storage (Cloudinary)
Fault/resolution photos are saved here permanently instead of on Render's disk (which wipes on every restart). Free — sign up at cloudinary.com, then: 1) copy your Cloud Name from the dashboard, 2) go to Settings → Upload → Add upload preset → set Signing Mode to "Unsigned" → copy its name. Until this is set up, photos still work but aren't permanent.
